### 8번. 값 전달 호출 결과 (2점)
다음 코드를 보고 물음에 답하시오.

```c
void swap_by_value(int x, int y)
{
    int temp = x;
    x = y;
    y = temp;
}

void swap_by_address(int *px, int *py)
{
    int temp = *px;
    *px = *py;
    *py = temp;
}
```

`int a = 10, b = 20;`일 때, `swap_by_value(a, b);` 호출 직후 `a`, `b`의 값을 쓰시오.
